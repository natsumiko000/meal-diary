# meal-diary

## サイト概要
その日の食事を10食品群＋主食項目に分けて記録することで、栄養バランスを管理できる

### サイトテーマ
日々の食事から健康を考える

### テーマを選んだ理由
栄養バランスの良い食事を以前より心がけており、食事から健康管理できるツールを作りたいと思ったから

### ターゲットユーザ
栄養バランスを見直したい人、健康意識を高めたい人

### 主な利用シーン
食事後、食事の内容を考える時
